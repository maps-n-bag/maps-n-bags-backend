# User

## [POST] Create User
```
/api/user
```
Create a new user.

### Endpoint

### Auth
- Not Required

### Body Parameters
```json
{
    "username": "niananto",
    "first_name": "Nazmul",
    "last_name": "Ananto",
    "email": "nazmulislamananto@gmail.com",
    "password": "something",
    "profile_pic": "https://picsum.photos/200",
    "cover_pic": "https://picsum.photos/200"
}
```

```json
{
    "id": 27,
    "username": "niananto",
    "first_name": "Nazmul",
    "last_name": "Ananto",
    "email": "nazmulislamananto@gmail.com",

}
```

# Plan
## [POST] Create Plan
```
/api/plan
```
Create a new travel plan.

### Endpoint

### Auth
- Required
- Not Required [public]

### Body Parameters
```json

    "start_date" : "13-07-2023",
    "end_date" : "17-07-2023",
    "regions" : [1],
    "tags" : [1, 5]
}
```

```json
{
    "id": 1,
    "user_id": 1,
    "title": "Cox's Bazar",
    "start_date": "2023-08-22T18:00:00.000Z",

    "description": "Cox's Bazar in 5 days",

}
```
